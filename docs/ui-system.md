# Visual Design System Framework

## Palette & Design Philosophy
- Theme Profile: High-end premium botanical field guide blended with professional architectural tools.
- Primary Colors: Muted natural tones (Stones, Sage Greens, Warm Creams).
- Typography Layout: High readability, elegant serif treatments for headings, clean sans-serif for inputs.
- Accessibility Guardrails: Enforce complete WCAG 2.1 AA contrast compliance across all custom templates.
